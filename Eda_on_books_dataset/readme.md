# 📚 Exploratory Data Analysis (EDA) on Books Dataset

## 📌 Project Overview

This project performs **Exploratory Data Analysis (EDA)** on a books dataset using Python. The objective is to clean the data, analyze important features, identify trends, detect outliers, and visualize insights using different plots.

The project demonstrates practical data analysis skills using **NumPy, Pandas, Matplotlib, and Seaborn**, making it suitable for data analyst and data science portfolios.

---

## 🚀 Features

- Data loading and inspection
- Handling missing values
- Data cleaning
- Descriptive statistics
- Outlier detection
- Distribution analysis
- Data visualization
- Dataset export after cleaning

---

- Pandas

## 📊 Analysis Performed

### ✅ Data Inspection

- Loaded dataset
- Checked data types
- Displayed dataset information
- Generated descriptive statistics
- Checked dataset shape

### ✅ Data Cleaning

- Handled missing values
- Filled missing book names with **"Unknown"**
- Filled missing language codes
- Exported cleaned dataset

### ✅ Exploratory Data Analysis

The notebook includes visualizations such as:

- 📈 Histogram of Publishing Year
- 📦 Box Plot for Sale Price (Outlier Detection)
- 🥧 Pie Chart for Author Rating Distribution

Additional insights can easily be extended for:

- Publisher analysis
- Language distribution
- Book pricing trends
- Revenue analysis

## 📈 Sample Insights

- Most books in the dataset were published between **1990 and 2010**.
- Sale prices contain several outliers.
- Author ratings show varying distributions across books.
- Missing values were successfully handled before analysis.
